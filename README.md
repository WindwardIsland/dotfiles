# My Current Linux Setup
These are my dotfiles for my current Linux setup (currently using Arch Linux with spectrwm).

## Why does this repository exist?
This repository exists in order to provide a way for easy access to my dotfiles. All the dotfiles I use for most of the programs I use can be found here. In addition, you are free to copy and use any of my dotfiles, as some of them are just copied from other people's dotfiles, but don't try to copy _all_ my dotfiles, as some of them are specific to my needs and might not be that well suited for you.

## Where can I find specific dotfiles?
The dotfiles that I use for most of the programs that I regularly use can be found in the `.config` folder. Custom scripts that I use can be found in the `.local/bin` folder. Configurations for the current browser that I use (librewolf) can be found in the `.librewolf` folder.
